# Contributing

Thanks for your interest in contributing!

## Before You Start

- **Check existing issues** — browse [open issues](https://github.com/devnchill/Asciify/issues) to see if someone already reported or is working on what you have in mind.
- **Get assigned** — if an issue already exists, comment on it to ask for assignment so two people don't work on the same thing.
- **No issue yet?** — create one first. Describe what you want to do so there's a place for discussion.

## Pull Requests

- **One PR per issue** — a PR should focus on a single issue. If it does many things at once, it may be closed.
- **Reference the issue** — link the issue in your PR description (e.g. `Closes #123`).
- **AI-assisted contributions are welcome** — if you used AI tooling to generate or assist with your code, please mention it in the PR description.
